# Jonathan Delgado Portfolio

A self-contained portfolio website inspired by the light, playful visual style of `hakk.tihupe.ee/en/`.

The page presents Jonathan David Delgado's EdTech, immersive learning, innovation, project, and experience content. Open `index.html` in a browser to view it, or use the local server URL when it is running.

## Files

- `index.html` - page content and structure
- `styles.css` - responsive visual design
